# AGENTS.md

## 目的
AI エージェント共通の作業方針を定義します。

## 基本方針
- **会話言語**: 日本語
- **コメント言語**: 日本語
- **エラーメッセージ言語**: 英語
- **コミット規約**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/)（description は日本語）

## 判断記録のルール
判断が必要な変更を行う場合は、判断内容、代替案、採用理由、前提条件、不確実性を明示してください。

## 開発手順（概要）
1. **プロジェクト理解**: ソースコードと構造を十分に調査する。
2. **依存関係インストール**: `dotnet restore` を実行する。
3. **変更実装**: 既存のスタイルと規約に従って実装する。
4. **テストと Lint/Format**: `dotnet format` を実行し、動作確認を行う。

## セキュリティ / 機密情報
- Discord Webhook URL などの認証情報をコミットしない。
- ログに機密情報を出力しない。

## リポジトリ固有
- このプロジェクトは VRCX (VRChat ログ管理ツール) のデータベースに依存しています。
- Windows 環境での動作を前提とした .NET 9.0 WinForms アプリケーションです。
